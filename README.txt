BUDGET MANAGER : DESCRIPTION

This application is designed to help users keep track of income and expenses.

User expenses are input and stored in a hashmap according to a Purchase Category(food, clothes, entertainment, other) to value-set mapping. 

This budget manager contains functionality that allows users:

1. save and retrieve purchase histrory from file

2. Analyze purchases made via a sorting system by purchase expenses in
 ascending order.

3. Display and keep track of every income and expense.
